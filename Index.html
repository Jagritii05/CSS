Three ways to add CSS to HTML.

  1. Inline CSS: Write style for every single element seperately Not recommended.
  2. Internal CSS: Add style tag in head tag and write the selector and deceleration there. For smaller project.
  3. External CSS: Write CSS in external .css file. Ideal case.
